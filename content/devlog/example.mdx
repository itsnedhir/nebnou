Started from zero today. `create-next-app`, TypeScript, Tailwind — the usual.

The interesting part was setting up contentlayer2 to treat MDX files as typed data. Define a schema, point it at a folder, get back a fully-typed collection. It's a nice pattern.

## what got done

- Scaffolded the Next.js 14 project
- Configured Tailwind with Courier New as the global font
- Built the sidebar with grouped nav and active link highlighting
- Set up contentlayer2 with 6 document types
- List pages and dynamic `[section]/[slug]` routes

## what's next

The site renders but there's nothing real in it yet. Need to fill it with actual content — which is always the harder part.

```bash
npm run dev
# ready on http://localhost:3000
```
